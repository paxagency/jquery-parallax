# Jquery-Parallax
Basic Parallax

#Examples
```javascript
$("img").parallax({
	speed:30,	  //amount of animation radius in pixels
	//scale:1.2,  //image scale
	//mode:0,     //movement switch direction
	//delay:.5,   //delay animation in seconds
	//axis:"X"    //axis to animate
	//blur:3	  //blur amount added to image
});
```
